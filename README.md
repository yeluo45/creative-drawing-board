# 儿童创意画板 (Creative Drawing Board)

一款面向3-6岁幼儿的创意画画网页游戏。

**访问地址**：https://yeluo45.github.io/creative-drawing-board/

## 功能特性

- **自由画画** — 手指/鼠标绘制，支持多种鲜艳颜色
- **10色配色盘** — 红、橙、黄、绿、蓝、紫、粉、棕、黑、白
- **3种笔刷大小** — 小/中/大，适应不同绘画需求
- **橡皮擦** — 轻松擦除绘制内容
- **清空画布** — 一键重新开始（带渐隐动画）
- **泡泡游戏** — 点击"泡泡"按钮进入泡泡模式，点击屏幕吹泡泡，支持连续生成和点击爆炸效果
- **音效反馈** — 所有交互都有愉快的音效（Web Audio API 合成，无外部音频文件）

## 目录结构

```
creative-drawing-board/
├── index.html    # 完整游戏（单文件，无依赖）

└── README.md     # 本文档
```

## 技术实现

- 纯 HTML5 Canvas（单文件，约745行）
- 无外部依赖，无网络请求，完全离线可用
- 响应式布局，适配 PC 和平板
- Web Audio API 合成音效
- 触控事件支持（平板可用）

## 本地运行

直接在浏览器中打开 `index.html` 文件即可运行，无需任何服务器或构建步骤。
